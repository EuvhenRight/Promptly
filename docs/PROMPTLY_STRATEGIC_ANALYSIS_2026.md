# Promptly — Стратегічний аналіз та Go-to-Market план 2026

> **Призначення:** Глибокий аналіз для успіху онлайн-маркетплейсу, натхненний підходом Thrive Agency. Стратегічна дорожня карта для позиціонування Promptly в екосистемі ринку AI-промптів.

---

## Короткий огляд

**Promptly** — це маркетплейс AI-промптів з міцним технічним фундаментом (Next.js 15, Firebase, Stripe, Genkit AI, Replicate). Ринок зростає на 25–27% CAGR з прогнозом до **$7.92 млрд до 2032 року**. Платформа **життєздатна та може генерувати дохід**, але стикається з сильними конкурентами (PromptBase 190K+ промптів, FlowGPT $1.9M доходу). Успіх залежить від **нішевого фокусу**, **спільноти** та **видимості в AI-пошуку** (GEO у стилі Thrive).

### Пріоритет GEO (Generative Engine Optimization)

Трафік від LLM швидко зростає. Оптимізуйте під:

- **ChatGPT Search** — розмовні запити
- **Google AI Overviews** — контент у форматі питання, на рівні пасажів
- **Perplexity** — цитування джерел
- **Gemini** — семантичний, природномовний контент

**Структура контенту, яку віддають перевагу LLM:**

- Чіткі заголовки H2/H3 у форматі Q&A
- Чек-листи, плюси/мінуси, цифри
- FAQ з прямими відповідями
- HTML з серверної сторони (уникати лише JS-контенту)

## 2. Стратегія спільноти

### Поточний стан

- Сторінка спільноти з стрічкою, топ-креаторами, новими учасниками
- Функція Generate з AI-моделями (диференціатор)
- Коментарі до промптів
- Pro-план з приватним промптами

### Прогалини порівняно з FlowGPT / PromptBase

| FlowGPT | PromptBase | Promptly |
|---------|------------|----------|
| AI-чат + додатки | 190K промптів | **Generate** (унікальна функція) |
| Орієнтація на спільноту | Надійний бренд | Менша спільнота |
| Токен-based залученість | Підписка | Кредити + плани |

### Рекомендації

1. **Програма для креаторів** — запросити 50–100 сильних креаторів з інcentive (кредити, featured-розміщення)
2. **Discord/Slack** — активна спільнота для prompt engineering
3. **Щотижневі челенджі** — «Промпт тижня» з призами
4. **Creator spotlights** — пости в блозі/соцмережах про успішних продавців
5. **Реферальні винагороди** — кредити за залучення креаторів та покупців

## 5. Позиціонування на ринку

### Позиціонійне твердження

> **Promptly** — маркетплейс промптів, де можна генерувати, купувати та продавати — все в одному місці. Створений для креаторів, які хочуть спробувати перед покупкою та монетизувати свої найкращі промпти.

### Диференціація

1. **Generate first** — спробуйте AI-генерацію перед покупкою промптів
2. **Кредитна модель** — гнучкість використання vs. фіксовані підписки
3. **Європейська база** — GDPR-friendly, ціни в EUR
4. **Спільнота + маркетплейс** — поєднання FlowGPT та PromptBase

### Конкурентна матриця

```
                    │ Спільнота │ Генерація │ Довіра/Бренд │ Нішевий фокус
────────────────────┼───────────┼────────────┼──────────────┼───────────────
PromptBase          │   Низька  │    Ні      │    Висока    │    Широкий
FlowGPT             │  Висока   │    Так    │   Середня    │    Широкий
Promptly            │ Середня   │    Так    │    Низька    │    TBD ★
AIPRM               │   Низька  │    Ні      │   Середня    │  Маркетинг
```

## 9. Пріоритети фокусу

### Робити першим

1. **Заповнити маркетплейс** — «No prompts found» убиває конверсію. Засіяти 200+ промптів.
2. **Обрати нішу** — наприклад, «промпти для маркетингу та SEO» або «промпти для освіти».
3. **GEO-контент** — 10–20 сторінок під «найкращі [ніша] промпти», «AI-промпти для [use case]».
4. **Програма креаторів** — 20–50 креаторів з бонусами.

### Відкласти

- Мобільний додаток
- Складні аналітики
- Кілька мов (до PMF)
- Фічі Web3 / NFT

### Розглянути видалення/спрощення

- Надто складні фільтри до появи обсягу
- Pro-only фічі, які плутають free-користувачів
- Дублювання навігації (desktop vs mobile)

## 11. Критичний фікс: порожній маркетплейс

**Поточний живий сайт показує «No prompts found»** — це головний вбивця конверсії №1.

### План дій

1. **Бутстреп-контент (тижні 1–2):**

3. **Скрапінг / імпорт (якщо етично):**
   - Розглянути імпорт публічних промптів з GitHub, форумів

# Зведення можливостей додатку "Promptly"

Цей документ описує ключові функції, доступні для різних типів користувачів у додатку.

## Функції для звичайних користувачів

- **Перегляд та пошук промптів:**
  - **Стрічка промптів:** Інтерактивна стрічка у вигляді masonry-сітки.
  - **Фільтрація:** Можливість фільтрувати промпти за категоріями, тегами, AI-моделями, типом контенту та ціною.
  - **Сортування:** Сортування за новизною, популярністю, рейтингом та ціною.
  - **Пошук:** Повнотекстовий пошук за ключовими словами в назвах.

- **Взаємодія з промптами:**
  - **Сторінка промпту:** Детальний перегляд з описом, автором, зображеннями та відгуками.
  - **Завантаження та перегляд зображень:** Можливість завантажити зображення промпту або відкрити його на повний екран.
  - **Копіювання посилання:** Кнопка "Поділитися" для копіювання URL сторінки.
  - **"Лайки":** Можливість додати промпт в обране.

- **Покупка та доступ:**
  - **Кошик:** Додавання кількох промптів у кошик для однієї транзакції.
  - **Покупка через Stripe:** Пряма оплата за допомогою банківської картки.
  - **Покупка за кредити:** Використання внутрішньої валюти (кредитів) для покупки промптів.
  - **Історія покупок:** Перегляд усіх придбаних промптів, кредитів та підписок у профілі.
  - **Доступ до контенту:** Після покупки користувач отримує доступ до захищеного тексту промпту.

- **Профіль та спільнота:**
  - **Авторизація:** Вхід у систему через Google.
  - **Публічний профіль:** Кожен користувач має публічну сторінку зі своїми промптами, описом та статистикою.
  - **Особистий кабінет:** Розділ для керування налаштуваннями профілю, планами підписки, гаманцем та сповіщеннями.
  - **Коментарі та рейтинг:** Можливість залишати відгуки та ставити оцінку (5 зірок) тільки для придбаних промптів.
  - **Підписки:** Можливість підписуватися на інших авторів.

## Функції для адміністраторів

Адміністратори мають доступ до спеціальної панелі (`/admin`), яка надає повний контроль над платформою.

- **Дашборд:**
  - **Головний дашборд:** Оглядова статистика по доходах, продажах та рейтингах.
  - **Аналітика продажів:** Деталізована сторінка зі статистикою доходів, найпопулярнішими товарами та найкращими продавцями за різні періоди (24 години, 7 днів, 30 днів, весь час).

- **Керування контентом:**
  - **Керування промптами:** Створення, редагування та видалення будь-яких промптів.
  - **Керування довідниками:** Можливість додавати, редагувати та видаляти:
    - **Категорії**
    - **Теги**
    - **Типи контенту** (наприклад, "Images", "Video")
    - **AI-моделі** (наприклад, "Midjourney", "Gemini")
  - **Керування коментарями:** Модерація, редагування та видалення коментарів користувачів.
  - **Налаштування фону:** Завантаження та керування фоновими зображеннями для пошукового рядка на головній сторінці.

- **Керування користувачами та фінансами:**
  - **Керування користувачами:** Перегляд списку всіх користувачів, редагування їхніх профілів та зміна ролей (наприклад, надання прав адміністратора).
  - **Керування виплатами:** Перегляд та обробка запитів на виплату коштів від авторів промптів (статуси: `pending`, `approved`, `paid`, `rejected`).

- **Інструменти імпорту:**
  - **Скрапер:** Інструмент для автоматичного парсингу та імпорту промптів з інших сайтів (наприклад, PromptHero) для швидкого наповнення контентом.

# Огляд проекту Promptly

## Що це

**Promptly** — маркетплейс для пошуку, покупки та продажу AI-промптів. Користувачі переглядають промпти, фільтрують за категоріями/моделями/ціною, купують через Stripe або кредити, залишають коментарі та оцінки (після покупки).

## Ключові можливості

- **Стрічка промптів** — masonry-сітка з фільтрами (категорія, теги, модель, тип, ціна).
- **Пошук** — за ключовими словами та тегами.
- **Авторизація** — Google Sign-In через Firebase Auth.
- **Профілі** — публічні профілі, підписки (followers/following), аватар, обкладинка.
- **Особистий кабінет** — керування профілем, планами, сповіщеннями та гаманцем.
- **Гаманець та виплати** — користувачі можуть заробляти кредити з продажів та запитувати виплати.
- **Кошик і оплата** — Stripe Checkout (embedded), підтримка одного промпту, кошика, кредитів, підписок (Starter/Pro).
- **Коментарі та рейтинг** — коментарі на сторінці промпту; верифіковані відгуки з рейтингом тільки для купивших (або безкоштовних промптів).
- **Адмін-панель** — дашборд з аналітикою, управління промптами, категоріями, тегами, типами, моделями, користувачами, коментарями, фонами пошуку, продажами та виплатами; скрапер для імпорту промптів.
- **AI** — Genkit flow для підказки релевантних тегів за назвою/описом промпту.

Детальний опис усіх можливостей дивіться у [11-features-summary.md](11-features-summary.md).

## Стек

| Категорія | Технологія |
|-----------|------------|
| Framework | Next.js 15 (App Router, React 19) |
| Стилі | Tailwind CSS, shadcn/ui (Radix) |
| Backend | Firebase (Auth, Firestore, Storage), Stripe |
| AI | Firebase Genkit, Google GenAI |
| Деплой | Vercel / Firebase App Hosting |

## Структура репозиторію (верхній рівень)

- `src/app/` — маршрути, сторінки, API routes.
- `src/components/` — UI, layout, feature-компоненти.
- `src/firebase/` — ініціалізація Firebase, хуки, провайдери, робота з Firestore/Auth.
- `src/hooks/` — хуки для категорій, тегів, типів, моделей, стрічки промптів.
- `src/lib/` — утиліти, типи, Stripe, placeholder-дані.
- `src/ai/` — Genkit, flows (наприклад, suggest-relevant-tags).
- `docs/` — технічна документація.
- `scripts/` — seed-скрипти, синхронізація Stripe → Firebase.

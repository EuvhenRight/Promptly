# **App Name**: PromptVerse

## Core Features:

- Prompt Feed: Display a masonry grid of AI prompts, with filters for categories, AI model, and price range.
- Prompt Search: Enable users to search prompts using keywords and tags.
- User Authentication: Allow users to create accounts, log in, and manage their profiles.
- Prompt Rating: Implement a 5-star rating system for prompts, displaying the average rating on the prompt card and detailed page.
- Commenting System: Enable users to post text comments on prompt pages. Only purchasers can leave verified reviews with a star rating.
- Top Creators Leaderboard: Display a ranked list of top creators based on sales and ratings, with toggles for different time ranges (week/month/all time).
- Prompt Generation Tool: AI tool to automatically suggest relevant tags based on prompt title and description to improve discoverability.

## Style Guidelines:

- Primary color: Midnight Blue (#2C3E50) for a professional and sophisticated feel.
- Background color: Light Gray (#F0F4F8) for a clean and modern look.
- Accent color: Electric Purple (#A78BFA) for highlights and interactive elements.
- Headline font: 'Space Grotesk', a sans-serif for headlines and short amounts of body text; body font: 'Inter', a sans-serif for longer text.
- Use 'lucide-react' icons throughout the application.
- Implement a responsive 3-column layout on desktop using Tailwind Grid, and adapt to a 1-column layout on mobile.
- Use subtle animations and transitions to enhance the user experience.
